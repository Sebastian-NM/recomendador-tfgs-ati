# references/

Este directorio contiene los datos del repositorio TEC.

## tfg_ati.json

Todos los Trabajos Finales de Graduación de la Escuela de ATI del TEC.
Extraído via REST API de DSpace (repositoriotec.tec.ac.cr).

Para actualizar los datos, corrés el script `extraer.mjs` desde la raíz
del proyecto y reemplazás este archivo con el resultado.
